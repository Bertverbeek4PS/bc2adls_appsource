## Suggested Commands
- Build/package with VS Code AL extension: `AL: Package` (Ctrl+Shift+B if configured); publish with `AL: Publish` to target sandbox/container.
- CLI alternative: `alc.exe /project:<path>` (from AL compiler) or `Publish-BCContainerApp` via BcContainerHelper for container deploy.
- Git (PowerShell): `git status`, `git diff`, `git add <path>`, `git commit -m "msg"`.
- PowerShell basics: `Get-ChildItem`, `Set-Location`, `Get-Content`, `Select-String` for grep-like search.
